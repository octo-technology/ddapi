Libraries
========

Libraries of data science functions available on the Data Driver environment.

* `BDACore <http://datadriver-doc-ddapi.s3-website-eu-west-1.amazonaws.com/bdacore/>`_: main repository of data science functions. Functions available for the new API.
* `BDACoreV1 <http://datadriver-doc-ddapi.s3-website-eu-west-1.amazonaws.com/bdacorev1/>`_: deprecated repository of data science functions. Only available for V1 API.

Tools
=====

Tool available with the Data Driver environment:
 * `DDTools <http://datadriver-doc-ddapi.s3-website-eu-west-1.amazonaws.com/ddtools/>`_: command line tool to manage your DataDriver platform and projects.
